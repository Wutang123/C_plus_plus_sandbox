## How to Clone Repository:
- `git clone git@github.com:Wutang123/C_plus_plus_sandbox.git`

## How to Compile and Run C++ Code:
- Compile: `g++ {C++ file} -o {excutable name}`

        Example: `g++ hello_world.cpp -o hello_world`

- Run: `./{excutable name}`
        
        Example: `./hello_world`
